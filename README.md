# SFHS Preview

Reusable latest-candidate review slot for Single-File HTML Software.

## Stable preview URL

`https://falloutmule.github.io/sfhs-preview/`

## Policy

- This repository displays only the latest review candidate.
- `index.html` is replaced for each preview.
- Old candidate files are not archived here.
- Source-project Git commits preserve the authoritative history.
- A preview is not a production release or merge approval.

See `candidate-manifest.json` for the currently displayed candidate identity.
